# Research: Investigate Change in Velocity Dispersion of ISO

This repo contains the codebase for my personal research into interstellar objects using the `lbparticles` python package.

The `lbparticles` python package's source is located (here)[https://github.com/LBParticles/LBParticles]

# Run Instructions

* Create a pipenv in the repo requirements.txt

* To construct all the figures for the research, run `python project.py`.
